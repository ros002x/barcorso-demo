# Caffè del Corso — demo

Sito statico dimostrativo per il Caffè del Corso di D'Arino Mario, Nova Siri.

## Avvio locale

Aprire `index.html` nel browser oppure servire la cartella con un server HTTP statico.

## Pubblicazione

Il progetto non richiede build ed è compatibile con GitHub Pages. La pubblicazione va effettuata solo dopo la verifica del titolare su contenuti, orari, menu, immagini e dati fiscali.

## Fonti dei dati

- Sluurpy: indirizzo, orari indicativi, prodotti e recensioni aggregate.
- Wanderlog: denominazione, telefono, indirizzo, orari aggregati e recensioni.
- Visura.pro: denominazione del titolare, sede e classificazione dell'attività.
- Tripadvisor: indirizzo e categoria.

I prezzi non sono pubblicati perché non verificati. Gli orari possono cambiare e vanno confermati telefonicamente.
